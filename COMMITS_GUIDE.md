## Как сделать скриншот

1. **Общая история:** GitHub → Code → Commits → скриншот списка коммитов
2. **Один коммит:** Нажать на хэш коммита → скриншот страницы с diff и датой
3. **Терминал:** После `git log --oneline` — скриншот вывода
